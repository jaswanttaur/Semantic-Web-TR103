Day 7

Tools (extensions) used to enhance the website:
- W3Validator: A tool to check the validity of web documents against W3C standards.
- Axe DevTools: An accessibility testing tool that helps identify and resolve accessibility issues.
- Wave: A web accessibility evaluation tool that provides visual feedback on the accessibility of web content.

For checking font:
- WhatFont: A browser extension that identifies fonts used on web pages.

Hex code of color:
- ColorZilla: A browser extension that allows you to pick colors from web pages and get their hex codes.

How to know which technology is used:
- Wappalyzer: A browser extension that identifies technologies used on websites, such as frameworks, analytics tools, and content management systems.
